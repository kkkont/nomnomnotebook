# Recipe blog application project

## backend

### Navigate to the backend directory
```
cd back
```

### Install dependencies
```
npm install
```

### Run the project
```
npm start
```

or 
```
npm server.js
```

## frontend

### Navigate to the frontend directory
```
cd front
```

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
